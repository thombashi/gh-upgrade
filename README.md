# gh-upgrade

## Summary
`gh-upgrade` is a [gh](https://github.com/cli/cli/releases) extension that updates the `gh` and its extensions to the latest version.


## Installation
```
gh extension install thombashi/gh-upgrade
```


## Usage
```
sudo -E gh upgrade
```


## Supported environments
- Linux
- macOS
